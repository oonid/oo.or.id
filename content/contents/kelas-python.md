Itulah yang menjadi pertimbangan untuk membuat kelas baru lagi, pengenalan pemrograman untuk pemula dengan bahasa pengantar Python.

Mulai dari tahap pengenalan pemrograman, pengenalan bahasa pemrograman Python, sampai di bagian akhirnya ada pengenalan membuat website dengan menggunakan bahasa pemrograman Python di atas Google Cloud Platform (yang memiliki free-tier untuk tahap awal).

Setelah mempelajari semua bahan dari kelas python ini, selanjutnya bisa mencoba membuat website sendiri seperti contoh pada [artikel tentang web di.lingkari.com berikut](http://oo.or.id/content/python-web-sample-project-di-lingkari-com).

Sejumlah materi diambil dari buku: [open source book "think python"](http://oo.or.id/content/open-source-book-think-python)

Untuk pengguna windows yang kesulitan melakukan instalasi pip (python package manager), salah satu peserta kelas python, [Nurul Hanifa](https://www.instagram.com/nihanii) memberikan masukan untuk instalasi dari situs [berikut](http://www.lfd.uci.edu/~gohlke/pythonlibs/") [^12].

Materi untuk kelas ini tersedia sebagai berikut [^9][^14].
